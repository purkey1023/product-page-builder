# 상세페이지 디자인 규칙서 (Design Rules)

이 문서는 AI가 상세페이지를 생성할 때 참고하는 디자인 규칙입니다.
실제 한국 프리미엄 뷰티 브랜드(ANUA, PEPTOIR, 라운드랩 등)의 상세페이지를 분석하여 작성되었습니다.

## 1. 전체 구조

### 780px 이미지 시퀀스
- 스마트스토어/쿠팡 상세페이지는 780px 너비의 이미지를 세로로 쌓는 구조
- ANUA 기준: 10~14개 이미지, 총 높이 약 20,000~25,000px
- 각 이미지는 독립적으로 의미를 전달해야 함

### 섹션 순서 (ANUA 분석 기준)
1. 히어로 (제품 + 핵심 메시지)
2. 브랜드 철학/스토리
3. 핵심 장점 (3개, 좌우 교차)
4. 성분 상세 (이미지 + 설명)
5. 텍스처/제형 (매크로 이미지)
6. 임상 결과/수치
7. 사용 방법 (3단계)
8. 비주얼 배너 (감성 이미지)
9. 리뷰/후기
10. 제품 정보/스펙
11. CTA (구매 유도)

## 2. 이미지 사용 패턴

### 이미지 유형별 사용 위치
| 이미지 유형 | 사용 방식 | 사용 섹션 |
|-----------|---------|---------|
| 제품 누끼 | 엘리먼트 (contain) | hero, specs, cta |
| 모델컷 | 배경 또는 대형 엘리먼트 | hero 배경, benefits |
| 텍스처 매크로 | 풀폭 배경 | texture |
| 성분 이미지 | 카드 내 엘리먼트 | ingredients |
| 라이프스타일 | 좌우 교차 엘리먼트 | benefits |
| 추상 배경 | 섹션 배경 | banner, cta |

### 이미지가 배경으로 가는 섹션 (3개만)
- **texture**: 텍스처 이미지 풀폭 + 약한 오버레이
- **banner**: 어두운 오버레이 + 밝은 텍스트
- **cta**: 어두운 오버레이 + CTA 텍스트

### 이미지가 엘리먼트로 가는 섹션 (나머지)
- **hero**: 제품 이미지 가운데 배치
- **benefits**: 좌우 교차 (이미지 340x340 + 텍스트)
- **ingredients**: 카드 내 (이미지 220x260 + 텍스트)
- **specs**: 좌측 (이미지 280x400 + 우측 스펙 텍스트)

## 3. 색상 & 톤

### 밝은 톤 유지
- 전체적으로 밝고 따뜻한 톤 (PEPTOIR/ANUA 캠페인 수준)
- 배경: 밝은 베이지(#FAF7F2), 크림(#FFF8F0), 화이트(#FFFFFF)
- 어두운 배경은 banner/cta에서만 사용
- 이미지 생성 시: +0.3~0.7EV 밝은 노출

### 무드별 팔레트
각 무드는 3색 체계: 배경(bg) + 텍스트(text) + 포인트(accent)

## 4. 타이포그래피

### 한국어 + 영문 혼용
- 섹션 라벨: 영문 UPPERCASE (KEY BENEFITS, HOW TO USE 등)
- 제목/본문: 한국어
- 브랜드명: 영문
- letterSpacing: 영문 라벨은 3~5px, 한국어는 0

### 텍스트 배치 원칙
- 풀폭 텍스트: x:40, width:700 (좌우 40px 패딩)
- 가운데 정렬이 기본
- 제목과 본문 사이 구분선(accent 색, width:100, height:2)

## 5. 레이아웃 패턴

### 좌우 교차 레이아웃 (가장 중요)
```
섹션A: [이미지 좌(40~380)] [텍스트 우(420~740)]
섹션B: [텍스트 좌(40~380)] [이미지 우(400~740)]  ← 반전
섹션C: [이미지 좌(40~380)] [텍스트 우(420~740)]
```
이 패턴이 ANUA 상세페이지의 핵심. 단조로움 방지.

### 카드 레이아웃
- 카드 배경: shape rect, borderRadius 16, accent 배경색
- 카드 내부: 20px 패딩
- 카드 간 간격: 20px

### 수치 그리드
- 2x2 배열: 좌(x:40, w:340) / 우(x:420, w:340)
- 큰 숫자(36px, bold, accent) + 설명(13px, muted)

## 6. 카피라이팅

### 원칙
- 문제-시간 앵커링: "화장이 무너지는 오후 2시"
- 혜택 번역: "히알루론산이 수분을 끌어당겨 촉촉하게"
- 구체적 숫자: "98% 만족도", "2주 만에 효과 체감"
- 감성 묘사: 사용 순간의 오감 표현

### 섹션별 카피 가이드
- hero: 강렬한 헤드라인 (15~25자)
- benefits: "~해서 ~효과" 형태 (각 15~30자)
- ingredients: 소비자 언어로 성분 설명
- reviews: 실감나는 후기 (인증구매 배지)
- cta: 행동 유도 ("지금 시작하면, 2주 후가 다릅니다")

## 7. AI 이미지 생성 가이드

### 프롬프트 구조
```
[Subject] + [Lighting] + [Composition] + [Background] + [Style] + [Quality]
```

### 필수 태그
- "Korean beauty editorial photography"
- "bright warm soft lighting"
- "no text, no logos, no watermarks"
- "4K resolution, highly detailed, commercial grade"

### 카메라 스펙 (포토리얼리즘 극대화)
- 제품: Canon EOS R5, RF 100mm f/2.8L Macro
- 모델: Sony A7R V, 85mm f/1.4 GM
- 텍스처: Phase One IQ4, Schneider 120mm Macro
- 배경: Hasselblad H6D-400c

### 톤 가이드
- 색온도: 5500-6500K (따뜻함)
- 노출: +0.3~0.7EV (밝게)
- 배경: 항상 밝음 (어두운 배경 금지)
- 피부: dewy luminosity (글로우 있는 촉촉한 느낌)

## 8. ANUA 실제 분석 데이터 (2026년 기준)

### ANUA 색상 팔레트
- Primary Accent: `#07cdc7` (틸/시안)
- Discount Red: `#e14425`
- Background: `#f8f8f8`
- Text Black: `#000000`
- Border: `#d0d0d0`
- Badge Border-Radius: 25px (둥근 알약 형태)

### ANUA 타이포그래피
| 요소 | 크기 | 굵기 | 줄간격 |
|------|------|------|--------|
| 할인율 | 27px | 700 | 1.4 |
| 섹션 헤더 | 18px | 600 | normal |
| 본문 | 14px | 400 | normal |
| 라벨 | 14px | 500 | normal |
| 작은 텍스트 | 11px | 400 | normal |

### ANUA 간격 체계
- 요소 간 수직 간격: 12px
- 섹션 상단 패딩: 24px
- 수평 패딩: 10~14px
- 버튼 높이: 60px

### ANUA 이미지 시퀀스 구조 (product_no=490)
- 총 14개 이미지 (핵심 10개 + 보조 4개)
- 모든 이미지 너비: 780px
- 개별 이미지 높이: 1,108px ~ 4,535px
- 총 높이: ~23,535px
- 포맷: PNG 8장 + GIF 2장 (애니메이션 데모)

### ANUA 디자인 핵심 패턴
1. **교차 레이아웃**: 좌이미지+우텍스트 → 좌텍스트+우이미지 반복
2. **한 메시지 한 섹션**: 섹션당 하나의 명확한 메시지
3. **여백 활용**: 12px 간격 + 24px 섹션 패딩으로 시각적 여유
4. **굵기 대비**: 400~700 weight 변화로 계층 구조 형성
5. **액션 중심**: 60px 높이 풀폭 버튼으로 구매 유도

## 9. 한국 이커머스 상세페이지 업계 표준

### 권장 규격
- 데스크톱 너비: 860px (모바일 우선 한국 이커머스 표준)
- 최소 콘텐츠 길이: 3,300px+
- 섹션 최적 높이: 1,000~1,200px

### 필수 포함 요소
- 고품질 제품/서비스 이미지
- 명확한 가격 표시
- 고객 리뷰 및 평점
- 강력한 구매 CTA
- 추가 제품 정보 및 주의사항
